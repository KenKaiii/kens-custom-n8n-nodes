## ⚡ **Quick Start**

### **1. Clone & Setup**
```bash
# Clone the template
git clone https://github.com/KenKaiii/kens-custom-n8n-nodes.git my-awesome-nodes
cd my-awesome-nodes

# Install dependencies
npm install

# Test build
npm run build
```

### **2. Get n8n Running**
```bash
# Install n8n globally
npm install -g n8n

# Start n8n with tunnel
n8n start --tunnel
```

🌐 **n8n will open at:** `http://localhost:5678`
